# Quizmaster Flutter

Новый отдельный Flutter-проект (Android + iOS + Web) для тренировки по Excel-файлам.

## Что умеет

- Загрузка нескольких Excel файлов.
- Выбор выборки по имени файла (файл = компетенция).
- Случайный порядок вопросов и вариантов.
- Проверка ответа и показ правильного варианта.
- Режим "Тренировать ошибки".
- Локальное сохранение текущей сессии и истории попыток.

## Ожидаемые колонки в Excel

Нужны 5 колонок:

1. Вопрос
2. Правильный ответ
3. Неправильный ответ 1
4. Неправильный ответ 2
5. Неправильный ответ 3

Если имена колонок отличаются, приложение пытается сопоставить по алиасам.
Если не совпало, использует первые 5 колонок по позиции.
